## 장기말(피스)

- [ ] 말은 위치를 갖는다.
  - [ ] X축 (0-8), Y축(0-9)로 구현한다.
- [ ] 말은 팀을 갖는다.
  - [ ] 팀은 RED, BLUE로 나뉘어 진다.
- [ ] 말은 움직인다.
  - [ ] 다른 말들의 위치를 받아 최종적으로 갈 수 있는 경로를 선별한다.
- [ ] 말은 다른 말을 잡을 수 있다.
  - [ ] 말은 같은 팀을 잡지 못하고, 오직 다른 팀만 잡을 수 있다.
