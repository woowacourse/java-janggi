# java-janggi

장기 미션 저장소

# 기능 요구사항

## 보드 초기화 기능
- [x] 장기 기물을 정해진 개수만큼 생성할 수 있다.
  - [x] 궁은 블루팀/레드팀이 1개씩 가짐
  - [x] 사는 블루팀/레드팀이 2개씩 가짐
  - [x] 차는 블루팀/레드팀이 2개씩 가짐
  - [x] 포는 블루팀/레드팀이 2개씩 가짐
  - [x] 마는 블루팀/레드팀이 2개씩 가짐
  - [x] 상은 블루팀/레드팀이 2개씩 가짐
  - [x] 병은 블루팀/레드팀이 5개씩 가짐
- [x] 장기판을 생성할 수 있다.
  - [x] 블루팀은 남쪽, 레드팀은 북쪽에 배치
  - [x] 각 기물을 고정된 위치에 배치
