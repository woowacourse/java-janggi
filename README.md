# piece

- 좌표를 가진다
    - 경로로 움직임을 표현한다
    - 해당 피스가 움직임이 가능한 모든 경로에 대해서 이동 가능한지 계산한다
    - 이동 기능을 가진다

# 장기판

- 장기판에 말의 위치는 리소스파일로 관리된다
  좌표는 (r,c) 0,0 ~ 8,9

## 유저 기능

- 유저는 마와 상을 바꿀수 있다(1.2에서는 구현X)
- 현재 차례를 출력한다 ex)(청,백)
- 현재 보드판을 출력한다
- 이동할 기물을 선택하고 이동시킬 수 있다
