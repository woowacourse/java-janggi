# 🚀 장기 미션


### 📌 기능
- 게임 시작 시 전체 기물들과 장기판을 올바른 위치에 초기화 한다.
- 기물
  - 기물의 팀(청, 홍)을 정한다
  - 기물의 좌표를 저장한다
- 보드
  - 좌표에 위치한 기물을 저장한다
  - 마와 상 간에는 배치를 변경할 수 있다.
    - 궁(General)
      - 청: 95
      - 홍: 25
    - 차(Chariot)
      - 청: 01, 09
      - 홍: 11, 19
    - 포(Cannon)
      - 청: 82, 88
      - 홍: 32, 38
    - 마(Horse)
      - 청: 02(03), 08(07)
      - 홍: 12(13), 18(17)
    - 상(Elephant)
      - 청: 03(02), 07(06)
      - 홍: 13(12), 17(18)
    - 사(Guard)
      - 청: 04, 06
      - 홍: 14, 16
    - 졸(Soldier)
      - 청: 71, 73, 75, 77, 79
      - 홍: 41, 43, 45, 47, 49
