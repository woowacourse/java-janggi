# 🚀 장기 미션


### 📌 기능
- 게임 시작 시 전체 기물들과 장기판을 올바른 위치에 초기화 한다.
- 기물
  - 기물의 팀(청, 홍)을 정한다
  - 기물의 좌표를 저장한다
- 보드
  - 좌표에 위치한 기물을 저장한다
    - 궁(宮) 
      - 청: 95
      - 홍: 25
    - 차(車)
      - 청: 01, 09
      - 홍: 11, 19
    - 포(包)
      - 청: 82, 88
      - 홍: 32, 38
    - 마(馬)
      - 청: 02, 08
      - 홍: 12, 18
    - 상(象)
      - 청: 03, 07
      - 홍: 13, 17
    - 사(士)
      - 청: 04, 06
      - 홍: 14, 16
    - 졸·병(卒·兵)
      - 청: 71, 73, 75, 77, 79
      - 홍: 41, 43, 45, 47, 49
